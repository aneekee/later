## Overview

Now users can only create/delete notes.

I want to:

- Differentiate resolved and unresolved notes, without deleting;
  - Be able to view only unresolved notes;
  - Be able to view only resolved notes;
  - Be able to view all notes;
- Encourage users into processing their notes after creation;

To do so I want to allow marking notes as resolved (+ add resolution notes for more clarity and tracking purposes).

I'll use the is_resolved kind of a flag for the filtering and processing analysys.
